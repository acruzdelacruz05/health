# Diseases in California From 1965-2002

by Antonio Cruz De La Cruz

This dataset, downloaded from [CORGIS](https://corgis-edu.github.io/corgis/csv/), contains information regarding the number of individuals detected with seven different diseases across the United States from 1928-2002. I've chosen to focus on the state of California and to analyze four of these diseases: Hepatitis A, Measles, Mumps, and Rubella.

Hepatitis A cases remains in the thousands according to the data from most recent year (2002).
